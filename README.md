Pomodoro Study Tracker

Este es un proyecto de Pomodoro desarrollado en Python diseñado para ayudar a gestionar el tiempo y mejorar la concentración. Su objetivo principal es evitar distracciones y permitirnos enfocarnos en actividades como repasar tareas, estudiar temas académicos o practicar programación.

Funcionalidades
Temporizador: Un contador regresivo para sesiones de estudio.

Study Tracking: Permite ingresar el nombre de la actividad que estás realizando.

Persistencia de Datos: Utiliza un archivo JSON para guardar y revisar el historial de las tareas completadas.

Interfaz Simple: Ejecución directa en la terminal (consola), facilitando su uso sin necesidad de instalaciones complejas.

Tecnologías Usadas
Python: Lenguaje de programación principal.

JSON: Formato de almacenamiento para el historial de actividades.

Time Library: Para el manejo de los segundos del temporizador.
